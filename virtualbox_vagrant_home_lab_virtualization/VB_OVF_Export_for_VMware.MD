

# ‚úÖ **Recommended Export Format**

When you export in VirtualBox:

**File ‚Üí Export Appliance (OVF)**

Choose:

### ‚úî **OVF 2.0**

**Do NOT use OVF 1.0** (VMware often rejects it or errors during import).

### ‚úî Disk Format: **VMDK**

VMware uses VMDK natively.
VirtualBox will suggest VMDK automatically‚Äîkeep that.

---

# ‚úÖ **Settings to Adjust Before Export**

### ### **1. Storage Controller**

VMware prefers:

* **SCSI (LSI Logic)**
  or
* **SATA**

But **avoid VirtualBox-only controllers**:

‚ùå ICH9
‚ùå VirtIO
‚ùå VBox internal controllers

So in the VM settings before exporting, change:

**Settings ‚Üí Storage ‚Üí Controller**
‚úî Use **SATA** or **SCSI (LSI Logic)**

This prevents import errors in VMware.

---

# ### **2. Remove or disable VirtualBox extensions/tools**

Inside the Windows Server VM:

* Uninstall **VirtualBox Guest Additions**
  (VMware Tools will replace it later)

Not mandatory for export but prevents driver/boot problems.

---

# ### **3. Network Adapter**

Set adapter to:

‚úî **Intel PRO/1000 MT Server**

VMware recognizes this automatically.

Avoid:
‚ùå Paravirtualized network (virtio-net)
‚ùå PCnet adapters

---

# ### **4. CPU and Chipset**

Recommended:

* Chipset: **PIIX3**
* CPU: **1‚Äì4 cores**
* Enable:
  ‚úî IO APIC
  ‚úî Hardware virtualization (VT-x/AMD-V)

Avoid:
‚ùå "KVM" or other paravirtualization modes

Use:
‚úî Paravirtualization Interface = **Default** or **None**

---

# ### **5. Export Options**

When VirtualBox asks for export settings:

### Set the following:

| Setting                   | Recommended Value                     |
| ------------------------- | ------------------------------------- |
| **Format**                | OVF 2.0                               |
| **Disk Format**           | VMDK                                  |
| **MAC Address Policy**    | Include MAC or Generate new (both ok) |
| **Manifest**              | Optional                              |
| **Appliance Compression** | OVA or OVF folder (your choice)       |

If you want a **single file**, choose **OVA**.

---
