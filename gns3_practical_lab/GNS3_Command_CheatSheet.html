GNS3 Cisco IOS Command Reference

Quick-start configuration and verification commands for Cisco vIOS-L2, vIOS-L3, 7200, 3725, and 1700 platforms running inside GNS3. Use this as an all-in-one lab aide for building and validating foundational routing and switching scenarios.

1. Platform Snapshots

vIOS-L2

Modern Layer 2 Catalyst-like switching. Full VLAN/STP suite, EtherChannel, and security features. Ideal for switching-focused labs.

vIOS-L3

Feature-rich Layer 3 routing with MPLS, VRFs, and advanced security. Use for up-to-date routing labs and control-plane automation.

7200

High-performance core router via Dynamips. Supports large routing tables, MPLS, QoS, and multi-interface WAN testing.

3725

Versatile enterprise edge router with NM-16ESW support for basic switching. Great for mixed routing/switching topologies.

1700

Branch-class router for CCNA-level labs. Limited resources but perfect for practicing fundamental routing protocols.

Best Practice: Use vIOS-L2 for switching labs instead of NM-16ESW when you need Rapid-PVST+, MST, or port-security features. Use vIOS-L3 or 7200 for routing-heavy scenarios requiring advanced protocols.

3. Interface IP Addressing

Router Interfaces (vIOS-L3, 7200, 3725, 1700)

conf t
interface GigabitEthernet0/0
 description Link to vIOS-L2
 ip address 192.0.2.1 255.255.255.0
 no shutdown
exit
interface Serial0/0
 ip address 10.0.0.1 255.255.255.252
 clock rate 64000
 no shutdown
end

SVI on vIOS-L2 (Switch Virtual Interface)

conf t
interface vlan 10
 ip address 192.0.2.10 255.255.255.0
 no shutdown
exit
ip default-gateway 192.0.2.1
end

Verification

  • show ip interface brief
  • show interfaces description
  • ping 192.0.2.10 source 192.0.2.1

4. VLANs, Access Ports & Trunks (Switching)

Create VLANs (vIOS-L2 / 3725 with NM-16ESW)

conf t
vlan 10
 name USERS
vlan 20
 name SERVERS
exit

Assign Access Ports

conf t
interface range GigabitEthernet0/1 - 0/10
 switchport mode access
 switchport access vlan 10
 spanning-tree portfast
exit

Configure 802.1Q Trunks

conf t
interface GigabitEthernet0/24
 description Uplink to vIOS-L3
 switchport trunk encapsulation dot1q
 switchport mode trunk
 switchport trunk native vlan 99
 switchport trunk allowed vlan 10,20,99
 spanning-tree portfast trunk
end

Verification

  • show vlan brief
  • show interfaces trunk
  • show spanning-tree vlan 10

5. EtherChannel (Port-Channel)

conf t
interface range GigabitEthernet0/11 - 0/12
 switchport mode trunk
 switchport trunk allowed vlan 10,20
 channel-group 1 mode active
exit
interface Port-channel1
 switchport mode trunk
 switchport trunk allowed vlan 10,20
end

Verification

  • show etherchannel summary
  • show interfaces port-channel 1 trunk
  • show spanning-tree interface port-channel 1
Platform Note: For legacy NM-16ESW modules on 3725/3745, use interface FastEthernetx/y and ensure the IOS image supports PAgP/LACP. vIOS-L2 supports both mode active (LACP) and mode desirable (PAgP).

6. Routing Protocol Configuration

RIPv2 (Best for 1700 & 3725 labs)

conf t
router rip
 version 2
 no auto-summary
 network 192.0.2.0
 network 10.0.0.0
 passive-interface GigabitEthernet0/0
end

Verification: show ip protocols, show ip rip database, show ip route rip

EIGRP (Classic Mode for 1700/3725, Named for vIOS-L3)

conf t
router eigrp 100
 network 10.0.0.0 0.0.0.3
 network 192.0.2.0 0.0.0.255
 passive-interface GigabitEthernet0/0
 no auto-summary
end

Verification: show ip eigrp neighbors, show ip eigrp topology, show ip route eigrp

OSPF (Process 1)

conf t
router ospf 1
 router-id 1.1.1.1
 network 192.0.2.0 0.0.0.255 area 0
 network 10.0.0.0 0.0.0.3 area 0
passive-interface GigabitEthernet0/0
end

Verification: show ip ospf neighbor, show ip ospf interface, show ip route ospf

BGP (Edge/7200/vIOS-L3)

conf t
router bgp 65001
 bgp log-neighbor-changes
 neighbor 203.0.113.2 remote-as 65002
 neighbor 203.0.113.2 update-source Loopback0
 network 198.51.100.0 mask 255.255.255.0
!
interface Loopback0
 ip address 198.51.100.1 255.255.255.0
end

Verification: show ip bgp summary, show ip bgp, show ip bgp neighbors

vIOS-L3 Enhancements: Enable address-family configuration for IPv6 or VPNv4 with router bgp → address-family modes. 7200 supports route-reflectors and MPLS when using advanced feature packs.
