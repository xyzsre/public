An open-source RDBMS, fork of MySQL.
2. Is MariaDB free?
Yes, open-source license.
3. Who maintains MariaDB?

4. What is the latest version?
Check mariadb.org.
5. Is MariaDB compatible with MySQL?
Yes, drop-in replacement.
6. What is a database?

7. What is SQL?
Query language for databases.
8. What is a table?
Rows and columns of data.
9. What is a row?
Single record in a table.
10. What is a column?
Field in a table.
11. What is a primary key?
Unique row identifier.
12. What is a foreign key?
Links tables via keys.
13. What is indexing?
Speeds up queries.
14. What is a view?

15. What is a stored procedure?
Precompiled SQL code.
16. What is a trigger?
Auto action on changes.
17. What is ACID?

18. What is normalization?
Reduces data redundancy.

20. What is a schema?
Database structure.
